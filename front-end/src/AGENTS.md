# Source Guide

This folder holds the app code.

Use:

- `views/` for route screens
- `components/` for reusable interface pieces
- `layouts/` for the shell
- `router/` for navigation
- `stores/` for shared client state
- `services/` for Finnhub requests
- `types/` for shared interfaces
- `plugins/` for app-wide setup

Keep the app simple and easy to demo.
