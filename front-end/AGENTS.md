# Frontend Guide

This folder contains the hackathon app that should be deployed.

Stack:

- Vue 3
- Vue Router
- Pinia
- Vuetify
- Finnhub API

Current focus:

- keep the UI simple
- keep the watchlist flow reliable
- keep GitHub Pages deployment working

Structure:

- `src/views/` for route pages
- `src/components/` for reusable UI
- `src/services/` for Finnhub access
- `src/stores/` for watchlist state
- `src/layouts/` for the app shell
